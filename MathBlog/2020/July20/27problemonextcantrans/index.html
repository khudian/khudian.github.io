One problem related with extended canonical transformations



A canonical transformation can be defined (a la Landau, Lifshitz) by generating function. This becomes important for extemded canonical transformations.

E.g. a function $W(q,Q,t,T)$ defines canonical transformation $$ (p,q,t)\mapsto (\rho,Q,T) \tag{1a} $$ such that $$ p_idq^i-H(p,q,t)dt=\rho_m dQ^m-h\left(\rho,Q,T\right)+dW\,, \tag{1b} $$ i.e. $$ \cases { p_i={\p W(q,Q,t,T)\over \p q^i}\cr \rho_m=-{\p W(q,Q,t,T)\over \p Q^m}\cr H(p,q,t)=-{\p W(q,Q,t,T)\over \p t}\cr h(\rho,Q,T)={\p W(q,Q,t,T)\over \p T}\cr } \tag{1c} $$ a function $S(q,Q,t)$ defines canonical transformation $$ p\mapsto \rho(p,q,t)\,,\quad Q\mapsto Q(p,q,t)\,,\quad \tag{2a} $$ such that $$ p_idq^i-H(p,q,t)dt=\rho_m dQ^m-h\left(\rho,Q,T\right)dT+dS\,, \tag{2b} $$ i.e. $$ \cases { p_i={\p S(q,Q,t)\over \p q^i}\cr \rho_m=-{\p S(q,Q,t)\over \p Q^m}\cr h(\rho,Q,T)=H(p,q,t)+{\p W(q,Q,t,T)\over \p T}\cr } \tag{2c} $$ and a function $G(q,Q)$ defines canonical transformation $$ p\mapsto \rho(p,q)\,,\quad Q\mapsto Q(p,q)\,, \tag{3a} $$ such that $$ p_idq^i=\rho_m dQ^m+dS\,, \tag{3b} $$ i.e. $$ \cases { p_i={\p G(q,Q)\over \p q^i}\cr \rho_m=-{\p G(q,Q,t)\over \p Q^m}\cr } \tag{3c} $$

QUESTION

How to define generating function $W(q,Q,t,T)$ such that transformation (1) coincides qith (2)?

E.g. if $S(q,Q,t)=G(q,Q)$ then (2) coincides with (3).

To answer this question we have to go to extended space $(p,q,t,E)$
