Essential parameters II

using book of Eisenhart



About one week ago we considered the concept of essential parameters for collection $\cal C$ of functions $f^i(x,a)$. [Usually collection is the span of the functions].

Recall that we say that set of $r$ parameters $a^\s$ ($s=1,\dots,r$) which describe the collection are not essential if we can reduce the number of parameters (see for detail blog on 02 August). We proved that $$ \hbox{\tt one can reduce the number of parameters describing the set of functions $\cal C$ } $$ $$ \hbox{\tt if and only if there exists a non-vanishing vector field $\R$ in the space } $$ $$ \hbox {\tt of parameters such that it does not change any function of the collection $\cal C $:}\,, $$ $$ \R=\varphi^\s(a){\p\over \p a^\s}\not=0\colon\, \hbox {such that for every function $f\in \cal C$}\,,\, df(\R)=\varphi^\s(a){\p f(x,a)\over \p a^\s}\equiv 0\,, \tag{1} $$

Remark Notice that equation (1) implies that ${\rm rank}{\p f\over \p a}< r$. However these conditions are not equivalent.



How to calculate the number of effective parameters? E.g. if collection $\cal C$ is the span of functions $\{f^i(x,a)\}$ then what is the relation of number of essential paramets and rank of Jacobian $\p f(x,a)\over \p a$?

Definition Let $\cal C$ be collection of maps from $\R^m$ to $\R^n$ depending on $r$ parameters $a^1,\dots,a^r$: $$ f^\mu=f^\mu(x^i,a^\a)\,,\quad (\mu=1,\dots,n\,,i=1,\dots,m\,, \a=1,\dots,r) \tag{2} $$ We say that $l$ ($l\leq r$) is the number of essential parameters if there exist $l$ parameters $A_I=A_I(a^1,\dots,a^r)$ ($I=1,\dots,l$) such that this $r$-parametric set of maps can be expressed as $l$-parametric set of maps: $$ f^\mu(x^i,a^\a)=F^\mu\left(x^i,A_I(a^\a)\right) \tag{4} $$ and these $l$ parameterrs are essential, i.e. ...

{\rm rank}M_2\leq\dots \leq M_k \tag{9} $$

Denote by $s_k$ rank of $M_k$.

Proposition 1) for every $k$, $s_k\leq s_{k+1}\leq r$

2) there exist $N_0$ such that for all $k\geq N_0$ $$ s_k=s_{N_0} \tag{10} $$

It is well defined $$ {\rm rank} M_\infty= \lim_{k\to\infty}{\rm rank} M_k \tag{10} $$





Theorem Number of essential parametrs of the family of maps $\{f^\mu(x^i,a^\a)\}$ is equal to the ${\rm rank M_\infty}$.

{\sl Proof} We have that ${\rm rank \,} M_0\leq r$. Suppose that ${\rm rank \,} M_0= r$. This means that rows of the matrix ${\p f^i\over \p a^\a}$ are linearly independent, i.e. $$ df(\R)=R^\a(a){\p f^i(x,a)\over \p a^\a}=0\Rightarrow R^\a\equiv 0\,. $$ This means that all parameters are essential. Thus we prove theorem in the case if ${\rm rank \,} M_0= r$.
