Moduli space, what is it







One of students asked me a question: Moduli spaces, What is it?

Suppose that we consider (usually infinite-dimensioanl) space, where we perform some identifications. We come by this identification to moduli space if the remaining degrees of freedom still take continuous values.

Consider examples.

First example

Consider a space of vector fields on $\R$. We identify two vector fields $\A=A(x){\p\over \p x}$ and $\C=C(x){\p\over \p x}$ if there exists diffeomorphism $F\colon \R\to\R$ which connects these fields, i.e. $$ \A=F_* \C\,,\quad\colon\,\,\, A(x){\d F(x)\over dx}=C(y)\big\vert_{y=F(x)}\,. \tag{1} $$ Exercise Consider vector fields $\A=x{d\over dx}$ and $\C=3y{d\over dy}$. One can see that map $$ y=x^3 $$ conneccts these fields, however this map is not!!! the diffeomorphism, and one can see that these vector fields cannot be identified by any diffeomorphism!

We see in this example that moduli space of vector fields on $\R$ with identification as above contains line:vector fields $\lambda x{d\over dx}$ and $\mu x{d\over dx}$ can be identified if and only if $\lambda=\mu$.
