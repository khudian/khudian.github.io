Let $M$ be a manifold with a point $\pb$ on it.
Let $\rho=\rho(\nu)$ be a formal density on manifold $M$
Remark The formal density $\rho(\nu)$ obeys also the condition that $$ \rho=e^{\varphi(\nu)}\rho_0(\nu)\,,\quad \hbox {where $\varphi(\nu)=\sum_{k\geq -1}\nu^k\varphi_k$}\,. $$ I do not know how to codify it without the use of the phase function Later I will call density of this type oscillatory density.
and $\sigma=\sigma(\nu)$ be a formal function on $M$ such that $$ {d\rho(\nu)\over d\nu}=\sigma(\nu)\rho(\nu)\,. \tag{1} $$ We consider formal generalised functions $\Lambda(\nu)$ with support at the point $\pb$ $$ \Lambda(\nu)=\Lambda_0+\nu \Lambda_1+\dots\,,\quad \hbox {and $\Lambda_0(1)\not =0$}\,. \tag{2} $$ We say that the generalised formal function $\Lambda$ is associated with the density $\rho$ if $$ \Lambda\left(\p_\v f+f\div_\rho \v\right)=0 \tag{3} $$ where $\v$ is an arbitrary vector field on $M$ (not formal) and $f$ is an arbitrary formal function on $M$.

We say that the generalised formal function $\Lambda$ is strongly associated with the density $\rho$ if this function is associanted with the density $\rho$ and the following condtion holds also: $$ {d\over d\nu}\Lambda(f)=-{n\over 2\nu}\Lambda+\Lambda\left({df\over d\nu}+ f\sigma\right)\,. \tag{4} $$ where $n$ is a dimension of manifold $M$, $f=f(\nu)$ is as usual an arbitrary formal function and a function $\sigma$ is defined by equation (1)
Equation (4) becomes natural if we apply it to generalised function $ \Lambda_{\rho_0,\varphi}=\nu^{-{n\over 2}}\int f\rho $, where a formal density $\rho$ is equal to $\rho=e^\varphi \rho_0$, $\rho_0$ is a density (non-vanishing at point $\pb$) and $\varphi$ is a formal function such that $ \varphi={1\over \nu}\varphi_{-1}+\varphi_0+\nu\varphi_1+\dots $, and a function $\varphi_1$ has non-degenerate hessian at the point $\pb$.

Notice that
a) generalised function associated with a formal density is defined up to a formal constant, and
b) generalised function stronlgly associated with a formal density is defined up to a constant (not formal just a complex number).
Theorem (Karabegov) If a density is an oscillatory density (see the footnote at the first page) then the inverse is also true.
