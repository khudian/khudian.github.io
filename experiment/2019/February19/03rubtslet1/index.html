This function defines a vector field $\X_H$ on the infinite-dimnensional space $C^\infty(M)$ of functions: $$ \X_H=\int dx H\left(x,{\p f\over \p x}\right){\delta\over \delta x}\,. \tag{1} $$ It is the vector field which defines at every `point'---function $f$ the infinitesimal curve $\gamma_{\X_H}$: $$ \gamma_{X_H}\colon \quad f\mapsto f+ \vare H\left(x^i, p_i={\p f\over \p x^i}\right)\,,\quad \vare^2=0\,. $$ The $P\exp\int_0^t \X_H d\tau$ of this vector field acting in the space of function sends any function $f=f_0(x)$ to the function $A(x,t)$ which is the solution of Hamilton-Jacobi differential equation: $$ A(x,t)\colon H\left(x,{\p A\over \p t}\right)=0\,\quad\hbox {with boundary condition}\,\, A\big\vert_{t=0}=f(x)\,. $$ This little bit peculiar point of view on the standard Hamilton Jacobi equation induces the following consequences.
